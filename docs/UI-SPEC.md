# 界面与交互规范（UI-SPEC）

> 5 个页面 + 横竖屏适配 + 交互细节 + 动效规范

---

## 一、设计语言

### 1.1 视觉基调

| 维度 | 规范 |
|---|---|
| 风格 | 卡通、柔和、大色块 |
| 主色 | 暖黄 `#FFC857`、薄荷绿 `#7BD389`、天蓝 `#5BA8E0` |
| 辅色 | 桃粉 `#FF9AA2`、淡紫 `#C8A8E9` |
| 中性 | 文字 `#2C3E50`、次文字 `#7F8C8D`、背景 `#FFFBF5`（暖米白） |
| 圆角 | 卡片 16px、按钮 20px（更大更柔和） |
| 阴影 | `0 4px 12px rgba(0,0,0,0.08)`（柔和） |
| 字体 | 系统字体栈（中文 PingFang SC / 苹方） |
| 字号基准 | 算式 80\~120px、按钮文字 36\~48px、正文 18\~22px |

### 1.2 配色对比度

所有文字与背景对比度 ≥ 4.5:1（WCAG AA）。

| 用途 | 配色 | 对比度 |
|---|---|---|
| 主文字 | `#2C3E50` on `#FFFBF5` | 12.6 : 1 ✅ |
| 次文字 | `#7F8C8D` on `#FFFBF5` | 4.7 : 1 ✅ |
| 正确反馈 | `#2D8C3C` on `#E8F8EC` | 5.5 : 1 ✅ |
| 错误反馈 | `#C03A2B` on `#FDECE9` | 5.7 : 1 ✅ |
| 按钮文字 | `#FFFFFF` on `#5BA8E0` | 4.6 : 1 ✅ |

### 2.3 CSS 适配策略

```css
/* 默认：竖屏 */
.expr { font-size: 80px; }
.options { display: grid; grid-template-columns: 1fr 1fr; }

/* 横屏 */
@media (orientation: landscape) {
  .expr { font-size: 120px; }
  .options { display: grid; grid-template-columns: repeat(4, 1fr); }
}

/* iPad mini 等较小设备 */
@media (max-width: 1024px) and (orientation: portrait) {
  .expr { font-size: 64px; }
}
```

## 四、组件规范

### 4.1 按钮（Button）

```css
.btn {
  min-width: 120px;
  min-height: 64px;       /* 适配儿童手指 */
  padding: 16px 24px;
  border-radius: 20px;
  font-size: 24px;
  font-weight: 600;
  transition: transform 0.1s ease, background 0.2s ease;
}

.btn:active {
  transform: scale(0.96);
}

.btn:disabled {
  opacity: 0.5;
  cursor: not-allowed;
}
```

**变体**：

- `.btn-primary` 主按钮（蓝色）
- `.btn-success` 成功（绿色）
- `.btn-ghost` 透明边框
- `.btn-danger` 重置等危险操作（红色）

### 4.4 倒计时进度条（TimeoutBar）

全题统一超时，倒计时进度条显示在算式上方。

**1 位数关卡**（5 秒）：

```
┌──────────────────────────┐
│ ▓▓▓▓▓▓▓▓▓░░░░░░░ 3/5s   │ ← 倒计时（暖黄）
└──────────────────────────┘
```

**2 位数关卡**（10 秒）：

```
┌──────────────────────────┐
│ ▓▓▓▓▓▓▓░░░░░░░░░ 4/10s  │ ← 倒计时
└──────────────────────────┘
```

```css
.timeout-bar {
  height: 8px;
  background: #F0EAD9;
  border-radius: 4px;
  overflow: hidden;
  margin-bottom: 16px;
}

.timeout-bar-fill {
  height: 100%;
  background: #5BA8E0;
  transition: width linear;
}

/* 剩 N 秒时变黄闪烁 */
.timeout-bar-fill.warn {
  background: #FFC857;
  animation: pulse 0.6s ease infinite;
}

@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.6; }
}
```

**倒计时行为**：

- 进度条从满到空，匀速缩减
- 剩 3 秒（1 位数剩 2 秒）：进度条变黄 + 闪烁
- 剩 1 秒：选项按钮整体轻微缩放（视觉提示）
- 超时：进度条变红，显示"没关系的~"反馈

## 五、动效规范

### 5.1 反馈动效

| 场景 | 动效 | 时长 | 实现 |
|---|---|---|---|
| 答对 | 按钮绿色高亮 + 缩放反馈（1 → 1.05 → 1） | 400ms | CSS keyframes |
| 答错 | 按钮左右抖动（±8px × 3 次） | 400ms | CSS keyframes |
| 切下一题 | 当前题淡出 + 下一题淡入 | 200ms | opacity transition |
| 通关 | 全屏星星汇聚 + 标题弹出 | 1500ms | 多元素 keyframes |
| 解锁下一关 | 锁图标变彩 + 缩放 | 600ms | transform + color |
| 成就弹出 | 从顶部滑入 + 缩放 + 短暂停留 + 滑出 | 3000ms | 完整动画 |
| 计时挑战结束 | 倒计时放大 + 缩放消失 | 500ms | transform |

### 5.2 减弱动效支持

```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

家长面板提供「动效」开关，覆盖系统设置。

### 5.3 关键 CSS 动画

```css
@keyframes wrongShake {
  0%, 100% { transform: translateX(0); }
  20%, 60% { transform: translateX(-8px); }
  40%, 80% { transform: translateX(8px); }

@keyframes correctPulse {
  0% { transform: scale(1); }
  50% { transform: scale(1.05); }
  100% { transform: scale(1); }
}

@keyframes confetti {
  0% { transform: translateY(0) rotate(0); opacity: 1; }
  100% { transform: translateY(400px) rotate(720deg); opacity: 0; }
}
```

## 六、反馈语料库（儿童友好）

避免命令式 / 评价式语气。

### 6.1 答错

- "再看看哦~"
- "差一点点~"
- "没关系，再想想~"
- "试试别的？"（错集加入时）
- "我们再来一次吧~"（错题优先重出时）

### 6.2 答对

- "太棒了！"（首次掌握新题）
- "你掌握啦！"（错题答对，从错集移除）

## 十、iPad 特定行为

### 10.1 安全区域

```css
.safe-top { padding-top: env(safe-area-inset-top); }
.safe-bottom { padding-bottom: env(safe-area-inset-bottom); }
```

### 10.2 防止误触

- 点击区域使用 `touch-action: manipulation`
- 关闭高亮 `-webkit-tap-highlight-color: transparent`
- 关闭双击缩放 `user-scalable=no`（仅在 viewport meta 中设置）

### 10.3 横竖屏切换

- 监听 `orientationchange` 与 `resize` 事件
- 重计算布局（CSS Grid + Flexbox 自适应）
- 当前关卡与连击不丢失

### 10.4 设备识别

```js
const isIPad = /iPad/.test(navigator.userAgent) ||
  (navigator.platform === 'MacIntel' && navigator.maxTouchPoints > 1);
const isIPhone = /iPhone/.test(navigator.userAgent);
```

iPhone 屏幕较小，UI 自动缩小（`@media (max-width: 500px)`）。
